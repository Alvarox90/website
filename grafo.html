Acerca del Proyecto

Este proyecto busca elaborar un programa con ineterfaz para poder visualizar algunas propiedades de un grafo

Ir a la página

Algunas Características

Recorrdio BFS

Mediante un sensor LM35 es capaz de detectar los gradps celcius en una pieza

Recorrido DFS

Implementadno un sensor de color RGB TCS34725

Agregaar Vertice y Arista

Para realizar esta tarea se utilizarón dos sensores tipo VL53LOX

Ver matriz de adyacencia

Se usó un potenciometro para obneter dichos datos

Resumen

Proyecto pensado para la industria de refrescos en la suepervición de latas.

Tecnologías Implementadas

Para la elaboración del sitio web se usó javascript, html y css. Se utilizarón Node ESP-12 E y ESP8266 para controlar los senosres.

Otras Tecnologías

PHP para tratar datos y uso de 000webhost como servidor
